macOS 전용 네이티브 앱

가장 우아한 방식의
마크다운 경험

Markdown Prism은 개발자와 작가를 위한 빠르고 가벼운 macOS 전용 마크다운 뷰어이자 에디터입니다. GFM, LaTeX 수식, Mermaid 다이어그램을 완벽하게 지원합니다.

macOS용 다운로드 GitHub에서 보기

생산성을 높이는 강력한 기능

Markdown Prism은 당신의 워크플로우를 방해하지 않으면서 필요한 모든 도구를 제공합니다.

실시간 미리보기

입력과 동시에 오른쪽 프리뷰 화면에서 결과물을 바로 확인하세요. 400ms 디바운싱으로 쾌적한 편집 경험을 제공합니다.

고급 렌더링

KaTeX를 통한 고품질 수식 렌더링과 Mermaid를 이용한 복잡한 다이어그램 및 순서도를 지원합니다.

구문 하이라이팅

180개 이상의 언어를 지원하는 highlight.js가 내장되어 있어, 어떤 코드 블록도 아름답게 표시합니다.

QuickLook 지원

앱을 열지 않고도 Finder에서 스페이스바를 눌러 마크다운 파일의 내용을 즉시 미리볼 수 있습니다.

오프라인 지원

모든 필수 라이브러리가 앱 내에 번들링되어 있어, 인터넷 연결 없이도 모든 기능을 완벽하게 사용할 수 있습니다.

파일 자동 감시

외부 에디터에서 파일을 수정하면 Markdown Prism이 이를 감지하고 실시간으로 내용을 업데이트합니다.

강력한 네이티브 에디터

단순한 뷰어를 넘어선 강력한 편집 기능을 제공합니다. macOS의 네이티브 텍스트 엔진을 활용하여 부드럽고 정확한 타이핑 경험을 제공하며, 마크다운 구문 강조를 통해 가독성을 높였습니다.

  • 부드러운 스크롤 및 타이핑
  • 자동 저장 및 버전 관리
  • 다크 모드 및 라이트 모드 지원
  • 표준 단축키 (Cmd+S, Cmd+O, Cmd+N)

수식부터 다이어그램까지

복잡한 논문 작업이나 기술 문서를 작성하시나요? Markdown Prism은 가장 까다로운 요구사항도 충족합니다.

  • GitHub Flavored Markdown 완벽 지원
  • 인라인 및 블록 수식 ($E=mc^2$)
  • Flowchart, Gantt, Sequence 다이어그램
  • 다크 모드 완벽 대응

설치 방법

몇 단계만으로 Markdown Prism을 설치하고 바로 사용할 수 있습니다.

Homebrew 설치

터미널에서 아래 명령어를 입력하여 간편하게 설치할 수 있습니다.

brew install hulryung/tap/markdown-prism
